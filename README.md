# Dungeon Synthesis

An implementation of the Wave Function Collapse Algorithm.

## Prerequisites

- .NET SDK 8
- Godot 4.5 (for DungeonVisualizer)

## Usage

- Run benchmarks: `dotnet run --project DungeonBenchmark -c Release`
- Test output: `dotnet run --project DungeonBenchmark -c Debug`

There are also run configurations for Jetbrains Rider (main IDE used).
